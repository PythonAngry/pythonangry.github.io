1.算法分析

变位词判断问题

a. 逐字检查 O(n^2)

  • 两重循环，外层遍历s1每个字符，将内层循环执行n次
  • 内层循环在s2查找字符，每个字符对比次数从1-N，各不相同
def anagramSolution1(s1,s2):
    alist = list(s2) #复制s2到列表
    pos1 = 0
    stillOK = True
    while pos1 < len(s1) and stillOK:
        pos2 = 0
        found = False
        while pos2 < len(alist) and not found: #循环s1每一个字符
            if s1[pos1] == alist[pos2]: #与s2逐个对比
                found = True
            else:
                pos2 = pos2 +1
        if found:
            alist[pos2] = None #找到，打勾设为空
        else:
            stillOK = False #没找到，失败
        pos1 = pos1 +1
    return stillOK

print(anagramSolution1('abcd','dcba'))

b. 排序比较 O(nlogn)

  • 本算法主导步骤是排序步骤
def anagramSolution2(s1,s2):
    alist1 = list(s1)
    alist2 = list(s2)
    
    alist1.sort()
    alist2.sort()
    pos = 0
    matches = True
    while pos < len(s1) and matches:
        if alist1[pos] == alist2[pos]:
                pos = pos +1
        else:
            matches = False
    return matches

print(anagramSolution2('abcde','edcba'))

c. 暴力法 O(n!)

  • 穷尽所有可能的组合
  • 将s1出现的字符进行全排列，查看s2是否出现在全排列列表中

d. 计数比较 O(n)

  • 3个非嵌套循环迭代
  • 依赖26计数器列表，储存空间用量最多，问题解法需要取舍权衡时间与空间
def anagramSolution4(s1,s2):
    c1 = [0] * 26 #26个0的列表
    c2 = [0] * 26
    for i in range(len(s1)): #分别计算
        pos = ord(s1[i]) - ord('a') #ord('a')=97 b98...
        c1[pos] = c1[pos] +1
    for i in range(len(s2)):
        pos = ord(s1[i]) - ord('a')
        c2[pos] = c2[pos] +1
    j = 0
    stillOK = True
    while j < 26 and stillOK: #计数器比较
        if c1[j] == c2[j]:
            j = j + 1
        else:
            stillOK = False
    return stillOK

print(anagramSolution4('apple','pleap'))
