0713 | Word培训

2020-07-14

不会吧不会吧都过了三个月了！！！
这是新工作提供的培训课程，感觉挺好。

  1. 最上一排叫选项卡（文件为菜单）

  2. 死机：自动保存恢复界面不出来，手动找回

    • 文件菜单 - 选项 - 保存

  3. 字体嵌入：别人电脑也可以看见特殊安装字体

    • 文件菜单 - 选项 - 保存 - 将字体嵌入文件

  4. 制表符与制表位

    • 开始 - 段落 - 显示/隐藏编辑标记
    • 左右侧缩进，首行缩进，悬挂缩进（除了第一行）
    • 段前段后间距，都设置，只有第一行及最后一行显现差别，中间仍为一行间距（PPT除外）
    • 视图 - 标尺：从上到下，首行缩进，悬挂缩进（除第一行），左缩进（全文本）
    • 全选想要改变格式的文章，标尺左上角改变制表符不同模式，在标尺点击所需位置，在文本所需位置点击Tab（格式刷仅刷格式，还是需要对所刷文本按Tab）
    • 去除制表符（只能替换删除）：ctrl+H - 替换 - 更多 - 特殊符号 - 制表符

  5. shift+回车：软回车

  6. 先文字后表格

    • 单独调整一个单元格：选中单元格（斜小黑箭头，选中为灰色）再拉动单元格边线
    • 表格斜下框线：选中单元格 - 表格工具 - 设计 - 边框 - 斜下框线
    • 文字转表格：选中需要转换的文本 - 插入 - 表格 - 文字转换成表格 - 选择分隔位置（全角符号直接复制粘贴至其他字符即可）
    • 表格转文本：表格工具 - 布局 - 文本转换为表格（文本内容调整，符号替换，格式清除）
    • 自定义表格样式：需要勾选基于该模板的新文档
    • 表格样式套用：保存新样式word文件为word模板格式 - 右键选项卡 - 自定义功能区 - 勾选开发工具 - 文档模板 - 管理器 - 选择所需样式模板文件 - 复制自定义模板到当前文件
    • 长表格重复标题行：选中重复标题行（可选多行） - 表格工具 - 布局 - 重复标题行
    • 表格跨页应用：选中跨页表格 - 段落 - 换行和分页 - 与下段同页 & 段中不分页

  7. 长文档排版

    • 显示隐藏样式：样式设置小符号 - 管理样式 - 推荐 - 选中标题4 - 显示
    • 修改同一样式：右键样式 - 全选（选择所有实例） - 修改（最好只对单一文档修改）
    • 若想对多个文档改样式：新建word模板，对所需样式进行修改 保存 - 到所需改样式的文档 - 开发工具 - 文档模板选用 - 勾选自动更新文档样式
    • 多级列表：段落 - 多级列表 - 定义新多级列表 - 更多 - 将各级别链接到样式 - 输入编号的格式 - 输入所需编号文字 - 选择此级别的编号样式 - 勾选正规形式编号（若删除了编号格式，下拉包含的级别编号来自，进行层及继承即可）
    • 插入目录：页面布局 - 分隔符 - 分节符 - 下一页 // 引用 - 目录 - 自定义目录 - 选项 - 选择要制作目录的级别 - 选择所需制表符前导符 - 如有更新 - 右键更新目录即可
    • 封面目录去除页码页眉页脚 - 双击正文页码 - 页眉和页脚工具 - 取消链接到前一条页眉 - 删去封面目录页码等 - 回到正文页 - 页眉和页脚工具 - 页码 - 设置页码格式 - 起始页码设为1（不再续前节，所以前面添加分节符比较好，只有分节符才能断开页码页眉页脚）
    • 新增页眉（页脚）样式：先编辑好所需页眉 - 全选页眉 - 页眉和页脚工具 - 文档部件 - 将所选内容保存到文档部件库 - 选择所需保存的部件库
    • 添加水印：设计 - 水印 // 双击页眉 - 插入图片 - 修改位置颜色 - 另存为PDF再发送出去~

  8. 邮件合并

    • 批量邮件生成：开始邮件合并 - 选择样式 - 选择收件人 - 使用现有列 - 插入合并域 - 完成并合并 - 编辑单个文档确认生成无误 - 发送电子邮件 - 收件人~邮箱 - 输入主题
    • 标签生成：选择excel文本前两行选择性粘贴，行列对调 - 新建word - 邮件 - 选择合适尺寸标签 - 粘贴文本 - 选择收件人 - 使用现有列 - 插入合并域 - 更新标签 - 完成并合并 - 编辑单个文档确认生成无误 - 打印

  9. 文档加密

    • 常用方法：文件另存为 - 工具 - 常规选项 - 选择文件加密选项（打开密码；修改密码）
    • 节保护：保护某一节 - 页面布局 - 添加连续分节符 - 审阅 - 限制编辑 - 仅允许在文档中进行此类型的编辑 - 选择填写窗体 - 选择所需要保护的节
    • 特殊的安全技术：仅允许在文档中进行此类型的编辑 - 只读 - 选择可编辑的内容 - 例外项勾选每个人 - 查找可编辑区域

  10. 文档修订

    • 比较文档：如果别人不知道有审阅修订功能，直接在文档修改 - 审阅 - 比较文档
